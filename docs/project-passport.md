# Паспорт проекта

## Общая информация

**Название проекта:** Cybersecurity Awareness Web App

**Тип проекта:** веб-приложение

**Тема:** просвещение пользователей по вопросам кибербезопасности

**Учебный контекст:** выпускная квалификационная работа

**Организация:** Тольяттинская академия управления

**Год:** 2026

## Цель проекта

Создать веб-приложение, которое помогает пользователям получить базовые знания
по кибербезопасности и применять их в повседневной цифровой среде.

## Задачи проекта

- подготовить образовательные материалы по ключевым темам кибербезопасности;
- реализовать удобный интерфейс для изучения материалов;
- добавить проверку усвоения знаний через вопросы или тесты;
- обеспечить понятную структуру приложения;
- подготовить проект к размещению в GitHub-репозитории;
- оформить техническую документацию для ВКР.

## Целевая аудитория

Пользователи без глубоких технических знаний, которым нужно разобраться в
базовых правилах безопасного поведения в интернете.

## Основные темы

- пароли и двухфакторная аутентификация;
- фишинг и социальная инженерия;
- защита персональных данных;
- безопасность электронной почты и мессенджеров;
- безопасная работа с файлами и ссылками;
- защита устройств и обновления программ;
- основы цифровой гигиены.

## Ожидаемый результат

Готовое веб-приложение с образовательными материалами и документацией, которое
можно использовать как демонстрационный и практический результат ВКР.
