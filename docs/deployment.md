# Запуск и размещение

## Локальный запуск

Точные команды запуска зависят от технологического стека проекта.

Если проект является статическим сайтом, его можно открыть через локальный
сервер:

```bash
python3 -m http.server 8000
```

После запуска открыть:

```text
http://localhost:8000
```

Если проект использует Node.js, типовой порядок запуска:

```bash
npm install
npm run dev
```

## Размещение

Веб-приложение можно разместить на:

- GitHub Pages, если проект статический;
- Vercel или Netlify, если используется frontend-фреймворк;
- обычном VPS или веб-хостинге, если требуется серверная часть.

## Контрольный список

- проект запускается локально;
- главная страница открывается без ошибок;
- все разделы доступны из интерфейса;
- тесты или проверочные вопросы работают корректно;
- документация актуальна;
- репозиторий не содержит лишних временных файлов и секретов.
